Wavefront OBJ Reader (objreader)

Reads Wavefront .OBJ files to produce polygonal and line data.
The OBJ reader reads data stored in Wavefront .OBJ format. The expected file extension is .obj, the datasets resulting from reading these files are polygons and lines.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the OBJ reader.

The value(s) must be a filename (or filenames).
