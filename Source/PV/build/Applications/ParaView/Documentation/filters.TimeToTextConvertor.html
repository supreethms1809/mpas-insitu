Annotate Time Filter (TimeToTextConvertor)

Shows input data time as text annnotation in the view.
The Annotate Time filter can be used to show the data time in a text annotation.
Property Description Default(s) Restrictions
Input This property specifies the input dataset for which to display the time.
Format The value of this property is a format string used to display the input time. The format string is specified using printf style. Time: %f
Shift The amount of time the input is shifted (after scaling). 0.0
Scale The factor by which the input time is scaled. 1.0
