AMR Contour (AMRDualContour)

Iso surface cell array.
Property Description Default(s) Restrictions
Input This property specifies the input of the filter.

Accepts input of following types:

  • vtkCompositeDataSet

The dataset much contain a field array (cell) with 1 component(s).

SelectMaterialArrays This property specifies the cell arrays from which the contour filter will compute contour cells.

An array of scalars is required.
Volume Fraction Value This property specifies the values at which to compute the isosurface. 0.1
Capping If this property is on, the the boundary of the data set is capped. 1

Accepts boolean values (0 or 1).
DegenerateCells If this property is on, a transition mesh between levels is created. 1

Accepts boolean values (0 or 1).
MultiprocessCommunication If this property is off, each process executes independantly. 1

Accepts boolean values (0 or 1).
SkipGhostCopy A simple test to see if ghost values are already set properly. 1

Accepts boolean values (0 or 1).
Triangulate Use triangles instead of quads on capping surfaces. 1

Accepts boolean values (0 or 1).
MergePoints Use more memory to merge points on the boundaries of blocks. 1

Accepts boolean values (0 or 1).
