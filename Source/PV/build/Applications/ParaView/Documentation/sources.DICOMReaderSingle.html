DICOM Reader (Single File) (DICOMReaderSingle)

Reads DICOM regular rectilinear grid data from a lone file.
DICOM is a file format often used in medical imaging. This version of the reader reads a single file (as opposed to a collection of files). The DICOM reader currently only supports uncompressed files.
Property Description Default(s) Restrictions
FileName DICOM file to load. This version of the reader only supports loading one file at a time.

The value(s) must be a filename (or filenames).
