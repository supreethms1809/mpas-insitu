proSTAR (STARCD) Reader (ProStarReader)

Reads geometry in proSTAR (STARCD) file format.
ProStarReader creates an unstructured grid dataset. It reads .cel/.vrt files stored in proSTAR (STARCD) ASCII format.
Property Description Default(s) Restrictions
FileName Set the file name for the proSTAR (STARCD) reader.

The value(s) must be a filename (or filenames).
