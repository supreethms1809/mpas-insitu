ImageResampling (ImageResampling)

Sample data attributes using a 3D image as probing mesh.
Property Description Default(s) Restrictions
Input This property specifies the dataset whose data will be probed

Accepts input of following types:

  • vtkDataSet
  • vtkCompositeDataSet
SamplingDimension How many linear resampling we want along each axis 10 10 10
UseInputBounds Do we use input bounds or custom ones? 1
