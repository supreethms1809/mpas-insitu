Enzo AMR Reader (EnzoReaderCore)

Read hierarchical box dataset from an Enzo file.
This Enzo reader loads data stored in Enzo format. The output of this reader is a hierarchical-box dataset.
Property Description Default(s) Restrictions
FileName This property specifies the Enzo file name

The value(s) must be a filename (or filenames).

NumBlocks
