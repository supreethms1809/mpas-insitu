spcth history reader (spcthhistoryreader)

Reads an spcth history file where each row translates into a single time step and the columns are points, materials and properties.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the VRML reader.

The value(s) must be a filename (or filenames).
CommentCharacter This property lists the characters that is used as the first character on comment lines %
Delimeter Character that is used as the delimeter. ,
TimestepValues Available timestep values.
