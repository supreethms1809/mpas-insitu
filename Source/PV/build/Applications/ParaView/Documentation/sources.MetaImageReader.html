Meta File Series Reader (MetaImageReader)

Reads a series of meta images.
Read a series of meta images. The file extension is .mhd
Property Description Default(s) Restrictions
FileNameInfo
FileNames The list of files to be read by the reader. Each file is expected to be in the meta format. The standard extension is .mhd. If more than one file is specified, the reader will switch to file series mode in which it will pretend that it can support time and provide one file per time step.

The value(s) must be a filename (or filenames).
TimestepValues Available timestep values.
