Transpose Table (TransposeTable)

Transpose a table.
Property Description Default(s) Restrictions
Input This property specifies the input to the filter.

Accepts input of following types:

  • vtkTable

The dataset much contain a field array (row) with 1 component(s).

Variables of Interest Choose arrays whose entries will be used to form observations for statistical analysis.
Add a column with original columns name This flag indicates if a column must be inserted at index 0 with the names (ids) of the input columns. 1

Accepts boolean values (0 or 1).
Use the column with original columns name This flag indicates if the output column must be named using the names listed in the index 0 column. 0

Accepts boolean values (0 or 1).
Only extract selected columns This flag indicates if the sub-table must be effectively transposed or not. 0

Accepts boolean values (0 or 1).
