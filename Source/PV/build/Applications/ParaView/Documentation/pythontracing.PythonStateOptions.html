PythonStateOptions (PythonStateOptions)

Options proxy used to prompt user for python state options.
Property Description Default(s) Restrictions
PropertiesToTraceOnCreate Select which properties to save in state 1

The value(s) is an enumeration of the following:

  • all properties (0)
  • any modified properties (1)
SkipHiddenDisplayProperties Skip displays properties/representations for elements hidden in a view. 1

Accepts boolean values (0 or 1).
