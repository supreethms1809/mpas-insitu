Line (LineSource)

This source creates a line between two points. The resolution indicates how many segments are in the line.
The Line source can be used to interactively (using a 3D widget) or manually (using the entries on the user interface) add a line to the 3D scene. The output of the Line source is polygonal data.
Property Description Default(s) Restrictions
Point1 This property controls the coordinates of the first endpoint of the line. -0.5 0.0 0.0
Point2 This property controls the coordinates of the second endpoint of the line. 0.5 0.0 0.0
Resolution This property specifies the number of pieces into which to divide the line. 6
