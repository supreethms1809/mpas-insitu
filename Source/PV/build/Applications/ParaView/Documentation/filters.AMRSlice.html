Slice AMR data (AMRSlice)

Slices AMR Data
This filter slices AMR data.
Property Description Default(s) Restrictions
Input This property specifies the input for this filter.

Accepts input of following types:

  • vtkOverlappingAMR
ForwardUpstream This property specifies whether or not requests will be propagated upstream. 0

Accepts boolean values (0 or 1).
EnablePrefetching This property specifies whether or not requests pre-fetching of blocks of the next level will be enabled. 0

Accepts boolean values (0 or 1).
Level Set maximum slice resolution. 0
OffSet Set's the offset from the origin of the data-set 1.0
Normal This property sets the normal of the slice. 0

The value(s) is an enumeration of the following:

  • X-Normal (1)
  • Y-Normal (2)
  • Z-Normal (3)
