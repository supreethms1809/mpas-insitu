EnSightWriter (EnSightWriter)

Write unstructured grid data as an EnSight file.
Writer to write unstructured grid data as an EnSight file. Binary files written on one system may not be readable on other systems. Be sure to specify the endian-ness of the file when reading it into EnSight.
Property Description Default(s) Restrictions
Input The input filter/source whose output dataset is to written to the file.

Accepts input of following types:

  • vtkUnstructuredGrid
