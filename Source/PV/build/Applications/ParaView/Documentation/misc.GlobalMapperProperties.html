GlobalMapperProperties (GlobalMapperProperties)

This proxy is useful for changing global mapper settings such as the coincident topology resolution setting on all rendering proceses.
Property Description Default(s) Restrictions
Mode none

The value(s) is an enumeration of the following:

  • Off (0)
  • PolygonOffset (1)
  • ShiftZBuffer (2)
PolygonOffsetParameters Used to set the polygon offset scale factor and units. Used when ResolveCoincidentTopology is set to PolygonOffset. These are global variables. none
PolygonOffsetFaces Used when ResolveCoincidentTopology is set to PolygonOffset. The polygon offset can be applied either to the solid polygonal faces or the lines/vertices. When set (default), the offset is applied to the faces otherwise it is applied to lines and vertices. This is a global variable. none

Accepts boolean values (0 or 1).
ZShift Used to set the z-shift if ResolveCoincidentTopology is set to ShiftZBuffer. This is a global variable. none
GlobalImmediateModeRendering Turn on/off flag to control whether data is rendered using immediate mode or note. Immediate mode rendering tends to be slower but it can handle larger datasets. The default value is immediate mode off. If you are having problems rendering a large dataset you might want to consider using immediate more rendering. none

Accepts boolean values (0 or 1).
