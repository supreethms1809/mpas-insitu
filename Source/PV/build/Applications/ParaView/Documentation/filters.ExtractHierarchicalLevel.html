Extract Level (ExtractHierarchicalLevel)

This filter extracts a range of groups from a hierarchical dataset.
This filter extracts a range of levels from a hierarchical dataset
Property Description Default(s) Restrictions
Input This property specifies the input to the Extract Group filter.

Accepts input of following types:

  • vtkUniformGridAMR
Levels This property lists the levels to extract from the input hierarchical dataset.
