SelectionSourceBase (SelectionSourceBase)

Internal proxy used to define the common API for Selection Source proxies. Do not use.
Property Description Default(s) Restrictions
FieldType The location of the array the selection came from (ex, point, cell). 0

The value(s) is an enumeration of the following:

  • CELL (0)
  • POINT (1)
  • FIELD (2)
  • VERTEX (3)
  • EDGE (4)
  • ROW (5)
ContainingCells When field type is POINT, this controls whether single vertex cells will be produced for each selected point, or whether the cells that contain each selected point will be produced. When field type is CELL this has no effect. 0

Accepts boolean values (0 or 1).
InsideOut When this property is false the selection describes everything that should be extracted. When this is true the selection describes everything that should NOT be extracted. 0

Accepts boolean values (0 or 1).
