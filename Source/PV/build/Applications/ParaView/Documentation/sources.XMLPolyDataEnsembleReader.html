XML PolyData Ensemble Reader (XMLPolyDataEnsembleReader)

Reads ensembles of VTK XML polydata files.
The XML Polydata reader reads ensembles of VTK XML polydata files. The standard extension is .evtp.
Property Description Default(s) Restrictions
MetaFileName The meta-file for the reader

The value(s) must be a filename (or filenames).
FileIndexRange Smallest and largest index for the list of simulations files. For instance, if we have 10 simulations the FileIndexRange is [0,9]. 0 0
FileIndex The index of the simulation file in the ensemble. 0
TimestepValues Reports time values to the GUI.
