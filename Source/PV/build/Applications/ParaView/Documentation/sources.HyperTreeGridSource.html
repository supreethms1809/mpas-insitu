Hyper Tree Grid (HyperTreeGridSource)

Hyper tree grid representing a tree-based AMR data set
This source uses input parameters, most notably a string descriptor, to generate a vtkHyperTreeGrid instance representing the corresponding a tree-based AMR grid with arbitrary rectilinear geometry and either binary or ternary subdivision.
Property Description Default(s) Restrictions
Dimension This property specifies the dimensionality of the hyper tree grid. 3
GridSize The three values in this property specify the number of root cells in each dimension of the hyper tree grid. 1 1 1
GridScale The three values in this property specify the scale of the root cells in each dimension of the hyper tree grid. 1 1 1
BranchFactor This property specifies the subdivision scheme (binary or ternary) of the hyper tree grid. 2
MaximumLevel The value of this property specifies the maximum number of levels in the hyper tree grid. 1
Descriptor This property specifies the string used to describe the hyper tree grid. .
