TimerLog (TimerLog)

This is a proxy used to control the timer log parameters on all processes. Note that since vtkTimerLog has loads of static ivar, most properties will affect all timer log instances.
Property Description Default(s) Restrictions
Enable Enables the timer-log on all processes. none

The value(s) is an enumeration of the following:
MaxEntries Set the maximum length for the log on all processes. none
