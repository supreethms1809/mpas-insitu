CompositeDataIDSelectionSource (CompositeDataIDSelectionSource)

CompositeDataIDSelectionSource used to create an ID based selection for composite datasets (Multiblock or HierarchicalBox dataset).
Property Description Default(s) Restrictions
IDs The list of IDs that will be added to the selection produced by the selection source. This takes 3-tuple of values as (flat-index, process number, id). 0 0 0
