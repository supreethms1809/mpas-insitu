XMLPUnstructuredGridWriter (XMLPUnstructuredGridWriter)

Write unstructured grid in a xml-based vtk data file.
Writer to write unstructured grid in a xml-based vtk data file. Can be used for parallel writing.
Property Description Default(s) Restrictions
Input The input filter/source whose output dataset is to written to the file.

Accepts input of following types:

  • vtkUnstructuredGrid
