OrderedCompositeDistributor (OrderedCompositeDistributor)

Property Description Default(s) Restrictions
Input Set the input to the Ordered Composite Distributor filter.
PassThrough Toggle whether to pass the data through without compositing. 0

Accepts boolean values (0 or 1).
PKdTree Set the vtkPKdTree to distribute with.
OutputType When not empty, the output will be converted to the given type.
