ENZO AMR Particles Reader (EnzoParticlesReader)

Reads AMR particles from an ENZO dataset
The Enzo particles reader loads particle simulation data stored in Enzo HDF5 format. The output of this reader is MultiBlock dataset where each block is a vtkPolyData that holds the particles (points) and corresponding particle data.
Property Description Default(s) Restrictions
FileNameInfo
FileNames The list of files to be read by the reader.

The value(s) must be a filename (or filenames).
TimestepValues Available timestep values.
