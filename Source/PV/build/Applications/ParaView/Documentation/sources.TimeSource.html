Time Source (TimeSource)

Produces a single cell uniform grid with data values that vary over a sin(t) wave from t=0 to t=1 (radian).
Produces a single cell uniform grid with data values that vary over a sin(t) wave from t=0 to t=1 (radian).
Property Description Default(s) Restrictions
Analytic Makes the time source produce discrete steps of or an analytic sin wave. 0

Accepts boolean values (0 or 1).
X Amplitude Controls how far the data set moves along X over time. 0.0
Y Amplitude Controls how far the data set moves along Y over time. 0.0
Growing Makes the time source grow and shrink along Y over time. 0

Accepts boolean values (0 or 1).
