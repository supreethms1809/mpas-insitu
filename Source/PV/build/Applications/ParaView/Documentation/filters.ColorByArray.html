Color By Array (ColorByArray)

This filter generate a color based image data based on a selected data scalar
Property Description Default(s) Restrictions
Input

Accepts input of following types:

  • vtkImageData

The dataset much contain a field array (point) with 1 component(s).

LookupTable
Color By This property specifies the name of the scalar array from which we will color by.

An array of scalars is required.

The value must be field array name.
RGBA NaN Color 0 0 0 255
OutputFormat 3

The value(s) is an enumeration of the following:

  • Luminance (1)
  • Luminance Alpha (2)
  • RGB (3)
  • RGBA (4)
