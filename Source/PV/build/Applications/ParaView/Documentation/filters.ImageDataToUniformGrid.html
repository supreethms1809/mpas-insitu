Image Data To Uniform Grid (ImageDataToUniformGrid)

Create a uniform grid from an image data by specified blanking arrays.
Create a vtkUniformGrid from a vtkImageData by passing in arrays to be used for point and/or cell blanking. By default, values of 0 in the specified array will result in a point or cell being blanked. Use Reverse to switch this.
Property Description Default(s) Restrictions
Input

Accepts input of following types:

  • vtkImageData

The dataset much contain a field array () with 1 component(s).

SelectInputScalars Specify the array to use for blanking.

An array of scalars is required.

The value must be field array name.
Reverse Reverse the array value to whether or not a point or cell is blanked. 0

Accepts boolean values (0 or 1).
