XML PolyData Reader (XMLPolyDataReaderCore)

Reads serial VTK XML polydata files.
The XML Polydata reader reads the VTK XML polydata file format. The standard extension is .vtp.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the VTK XML polygonal dataset reader.

The value(s) must be a filename (or filenames).
TimestepValues Available timestep values.
CellArrayInfo
Cell Arrays This property lists which cell-centered arrays to read.

The list of array names is provided by the reader.
PointArrayInfo
Point Arrays This property lists which point-centered arrays to read.

The list of array names is provided by the reader.
