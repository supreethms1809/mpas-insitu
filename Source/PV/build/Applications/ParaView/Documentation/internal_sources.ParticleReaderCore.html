VTK Particle Reader (ParticleReaderCore)

Core for ParticleReader.
Property Description Default(s) Restrictions
FileName The name of the file to load.

The value(s) must be a filename (or filenames).
ByteOrder Set/Get the byte ordering of the file you are trying to read in. 0

The value(s) is an enumeration of the following:

  • BigEndian (0)
  • LittleEndian (1)
HasScalar Default: 1. If 1 then each particle has a value associated with it. 1

Accepts boolean values (0 or 1).
FileType Set the file type. The options are: - Unknown (0) (default) the reader will attempt to determine the file type. If this fails then you should set the file type yourself. - Text (1) the file type is text. - Binary (2) the file type is binary. 0

The value(s) is an enumeration of the following:

  • Unknown (0)
  • Text (1)
  • Binary (2)
DataType Get/Set the data type. The options are: - Float (10) (default) single precision floating point. - Double (11) double precision floating point. 10

The value(s) is an enumeration of the following:

  • Float (10)
  • Double (11)
