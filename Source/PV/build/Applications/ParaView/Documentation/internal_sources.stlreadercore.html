STL Reader (stlreadercore)

Reads ASCII or binary stereo lithography (STL) files.
The STL reader reads ASCII or binary stereo lithography (STL) files. The expected file extension is .stl. The output of this reader is a polygonal dataset.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the STL reader.

The value(s) must be a filename (or filenames).
ScalarTags When set to true, the reader will include cell data describing which solid a cell belongs to. 1

Accepts boolean values (0 or 1).
