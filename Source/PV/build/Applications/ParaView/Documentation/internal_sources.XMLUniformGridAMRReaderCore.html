XML UniformGrid AMR reader (XMLUniformGridAMRReaderCore)

Property Description Default(s) Restrictions
DefaultNumberOfLevels This reader supports demand-driven heavy data reading i.e. downstream pipeline can request specific blocks from the AMR using vtkCompositeDataPipeline::UPDATE_COMPOSITE_INDICES() key in RequestUpdateExtent() pass. However, when down-stream doesn't provide any specific keys, the default behavior can be setup to read at-most N levels by default. The number of levels read can be set using this method. Set this to 0 to imply no limit. Default is 0. 1
