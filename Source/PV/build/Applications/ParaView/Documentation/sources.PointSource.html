Point Source (PointSource)

Create a point cloud of a certain size, radius, and center.
The point source creates a specified number of points within a given radius about a specified center point.
Property Description Default(s) Restrictions
Center This property specifies the 3D coordinates of the center of the point cloud. 0.0 0.0 0.0
NumberOfPoints This property specifies the number of points in the point cloud. 1
Radius This property specifies the radius of the point cloud, measured from the value of the Center property. 0.0
