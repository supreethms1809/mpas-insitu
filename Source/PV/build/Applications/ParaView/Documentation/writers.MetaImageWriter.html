MetaImageWriter (MetaImageWriter)

Write a binary UNC meta image data.
Writer to write a binary UNC meta image data. This is a fairly simple yet powerful format consisting of a text header and a binary data section. MetaImage headers are expected to have extension: ".mha" or ".mhd"
Property Description Default(s) Restrictions
Input The input filter/source whose output dataset is to written to the file.

Accepts input of following types:

  • vtkImageData
FileName The name of the file to be written.
