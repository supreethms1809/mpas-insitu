Ruler (Ruler)

This is a line source that can be used to measure distance between two points
The ruler can be used to interactively (using a 3D widget) or manually (using the entries on the user interface) specify two points and then determine the distance between the two points. To place points on the surface of any dataset, one can use the 'p' key shortcut.
Property Description Default(s) Restrictions
Point1 This property controls the coordinates of the first endpoint of the line. -0.5 0.0 0.0
Point2 This property controls the coordinates of the second endpoint of the line. 0.5 0.0 0.0
