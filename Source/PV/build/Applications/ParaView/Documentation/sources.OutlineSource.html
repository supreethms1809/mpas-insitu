Outline (OutlineSource)

3D outline of the specified bounds.
The Outline source creates an axis aligned bounding box given the user-specified minimum and maximum coordinates for each axis.
Property Description Default(s) Restrictions
Bounds The values of this property specify the minimum and maximum X, Y, and Z coordinates (X min, X max, Y min, Y max, Z min, Z max) for drawing the outline. 0 1 0 1 0 1
