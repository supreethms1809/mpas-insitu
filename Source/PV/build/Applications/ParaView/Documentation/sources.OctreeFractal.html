Octree Fractal (OctreeFractal)

Test source for octree with Mandelbrot fractal
Create an octree from a Mandelbrot fractal. See the Mandelbrot source for a description of the variables used.
Property Description Default(s) Restrictions
Dimension This property specifies the dimensionality of the fractal: 1D - Binary tree line, 2D - Quadtree plane, 3D - Octree volume. 2
MaximumLevel This property specifies the maximum refinement level for the grid. 5
MinimumLevel This property specifies the minimum refinement level for the grid. 3
ProjectionAxes This property indicates which axes of the dataset to display. See Mandelbrot source for a description of the possible axes. 0 1 2
OriginCX This property specifies the imaginary and real values for C (constant) and X (initial value). See Mandelbrot source for a description of the C and X variables. -1.75 -1.25 0.0 0.0
SizeCX The four values of this property indicate the length of the output in each of the four dimensions (the real and imaginary components of C and the real and imaginary components of X). The three dimensions specified in the Projection Axes property will determine which of these values specify the length of the axes in the output. 2.5 2.5 2.0 1.5
Maximum Number of Iterations The value of this property specifies the limit on computational iterations (i.e., the maximum number of iterations to perform to determine if the value will go above 2). Values less than 2.0 after the specified number of iterations are considered in the fractal set. 100
Threshold This property specifies a threshold value that determines when to subdivide a leaf node. 2.0
