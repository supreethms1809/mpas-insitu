XDMF Reader (XdmfReader)

Reads XDMF (eXtensible Data Model and Format) files.
The XDMF reader reads files in XDMF format. The expected file extension is .xmf. Metadata is stored in the XDMF file using an XML format, and large attribute arrays are stored in a corresponding HDF5 file. The output may be unstructured grid, structured grid, or rectiliner grid. See http://www.xdmf.org for a description of the file format.
