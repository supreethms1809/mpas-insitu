ADIOSReader (ADIOSReader)

Read ADIOS BP files using VTKs schema.
Reader to read ADIOS BP files using a VTK specific schema
Property Description Default(s) Restrictions
FileName This property specifies the ADIOS filename.

The value(s) must be a filename (or filenames).
TimestepValues
Read Method 0

The value(s) is an enumeration of the following:

  • BP (0)
  • BPAggregate (1)
  • DataSpaces (3)
  • DIMES (4)
  • FlexPath (5)
