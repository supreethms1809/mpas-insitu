FLASH Particles AMR Reader (FlashParticlesReaderCore)

Reads AMR particles from FLASH dataset
The Flash particles reader loads particle simulation data stored in Flash format. The output of this reader is a MultiBlock dataset where each block is vtkPolyData that holds the particles and corresponding particle data.
Property Description Default(s) Restrictions
FileName This property specifies the Enzo file name

The value(s) must be a filename (or filenames).
NumParticles
Frequency This property sets the frequency interval for loading particles. 1
FilterLocation Turns on filtering of particles by location 0

Accepts boolean values (0 or 1).
MinLocation This property sets the minimum 3-D coordinate location by which the particles will be filtered out. 0.0 0.0 0.0
MaxLocation This property sets the minimum 3-D coordinate location by which the particles will be filtered out. 0.0 0.0 0.0
ParticleArrayInfo
Particle Arrays This property lists which particle arrays to read.

The list of array names is provided by the reader.
