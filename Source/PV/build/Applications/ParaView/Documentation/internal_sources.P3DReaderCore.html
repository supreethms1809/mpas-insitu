P3DReaderCore (P3DReaderCore)

Reads ASCII or binary PLOT3D files.
PLOT3D is a plotting package developed at NASA. The PLOT3D reader can read both ASCII and binary PLOT3D files. The default file extension for the geometry files is .xyz, and the default file extension for the solution files is .q. The output of this reader is a multi-block dataset containing curvilinear (structured grid) datasets.
Property Description Default(s) Restrictions
FileName This property specifies the .xyz (geometry) file name for the PLOT3D reader.

The value(s) must be a filename (or filenames).
QFileName This property specifies the .q (solution) file name for the PLOT3D reader.

The value(s) must be a filename (or filenames).
FunctionFileName This property specifies the .f (function) file name for the PLOT3D reader.

The value(s) must be a filename (or filenames).
AutoDetectFormat When this options is enabled, the reader will try to automatically figure out various settings for a binary file. The other options are ignored. Does not work for text (ascii) files. 1

Accepts boolean values (0 or 1).

Accepts boolean values (0 or 1).
MultiGrid This property indicates whether these files have multiple-block information. 0

Accepts boolean values (0 or 1).
HasByteCount This property indicates whether this file uses Fortran leading-trailing byte counts. 0

Accepts boolean values (0 or 1).
IBlanking This property indicates whether this dataset has visibility information. 0

Accepts boolean values (0 or 1).
TwoDimensionalGeometry This property indicates whether the data is two- or three-dimensional. 0

Accepts boolean values (0 or 1).
ForceRead This property indicates whether to attempt to read the file even if it appears corrupt 0

ByteOrder This property specifies the byte order of any binary file(s). 0

The value(s) is an enumeration of the following:

Functions This property lists which functions (point-centered arrays) to compute.

The value(s) is an enumeration of the following:

  • Scalar - Pressure (110)
  • Scalar - Temperature (120)
  • Scalar - Enthalpy (130)
  • Scalar - Internal energy (140)
  • Scalar - Kinetic energy (144)
  • Scalar - Velocity magnitude (153)
  • Scalar - Entropy (170)
  • Scalar - Swirl. (184)
  • Vector - Velocity (200)
  • Vector - Vorticity (201)
  • Vector - Pressure gradient (210)
