XMLMultiBlockDataWriter (XMLMultiBlockDataWriter)

Write a multiblock in a xml-based vtk data file.
Writer to write a multiblock dataset in a xml-based vtk data file. Can be used for parallel writing as well as serial writing.
Property Description Default(s) Restrictions
Input The input filter/source whose output dataset is to written to the file.

Accepts input of following types:

  • vtkMultiBlockDataSet
