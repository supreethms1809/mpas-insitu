ConvertSelection (ConvertSelection)

Converts a selection from one type to another.
Property Description Default(s) Restrictions
DataInput Set the vtkDataObject input used to convert the selection.

Accepts input of following types:

  • vtkDataObject
Input Set the selection to convert.

Accepts input of following types:

  • vtkSelection
OutputType Set the ContentType for the output. 5

The value(s) is an enumeration of the following:

  • SELECTIONS (0)
  • GLOBALIDs (1)
  • PEDIGREEIDS (2)
  • VALUES (3)
  • INDICES (4)
  • FRUSTUM (5)
  • LOCATION (6)
  • THRESHOLDS (7)
ArrayNames
MatchAnyValues 0

Accepts boolean values (0 or 1).
