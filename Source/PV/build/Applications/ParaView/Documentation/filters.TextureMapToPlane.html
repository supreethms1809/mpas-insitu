Texture Map to Plane (TextureMapToPlane)

Generate texture coordinates by mapping points to plane.
TextureMapToPlane is a filter that generates 2D texture coordinates by mapping input dataset points onto a plane. The plane is generated automatically. A least squares method is used to generate the plane automatically.
Property Description Default(s) Restrictions
Input Set the input to the Texture Map to Plane filter.

Accepts input of following types:

  • vtkDataSet
