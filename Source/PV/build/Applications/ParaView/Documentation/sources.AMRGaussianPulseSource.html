AMR GaussianPulse Source (AMRGaussianPulseSource)

Create AMR dataset w/ Gaussian Pulse
AMR dataset source, used for generating sample Berger-Collela AMR dataset with a Gaussian Pulse field at the center.
Property Description Default(s) Restrictions
Dimension Sets the desired dimension for the AMR dataset to generate. 3

The value(s) is an enumeration of the following:

  • 2D (2)
  • 3D (3)
Root Spacing Set the spacing at level 0. 0.5
RefinementRatio Sets the desired dimension for the AMR dataset to generate. 2

The value(s) is an enumeration of the following:

  • 2 (2)
  • 3 (3)
  • 4 (4)
XPulseOrigin Set x-coordinate for the pulse origin 0.0
YPulseOrigin Set y-coordinate for the pulse origin 0.0
ZPulseOrigin Set z-coordinate for the pulse origin 0.0
XPulseWidth Set x-coordinate for the pulse Width 0.5
YPulseWidth Set y-coordinate for the pulse Width 0.5
ZPulseWidth Set z-coordinate for the pulse Width 0.5
PulseAmplitude Sets the amplitude of the pulse 0.5
