Cone (ConeSource)

Create a 3D cone of a given radius and height.
The Cone source can be used to add a polygonal cone to the 3D scene. The output of the Cone source is polygonal data.
Property Description Default(s) Restrictions
Resolution This property indicates the number of divisions around the cone. The higher this number, the closer the polygonal approximation will come to representing a cone, and the more polygons it will contain. 6
Radius This property specifies the radius of the base of the cone. 0.5
Height This property specifies the height of the cone. 1.0
Center This property specifies the center of the cone. 0.0 0.0 0.0
Direction Set the orientation vector of the cone. The vector does not have to be normalized. The cone will point in the direction specified. 1.0 0.0 0.0
Capping If this property is set to 1, the base of the cone will be capped with a filled polygon. Otherwise, the base of the cone will be open. 1

Accepts boolean values (0 or 1).
