Table To Structured Grid (TableToStructuredGrid)

Converts to table to structured grid.
The TableToStructuredGrid filter converts a vtkTable to a vtkStructuredGrid. One must specifies the columns in the input table to use as the X, Y and Z coordinates for the points in the output, and the whole extent.
Property Description Default(s) Restrictions
Input This property specifies the input..

Accepts input of following types:

  • vtkTable

The dataset much contain a field array (row) with 1 component(s).

WholeExtent 0 0 0 0 0 0
XColumn This property specifies which data array is going to be used as the X coordinate in the generated polydata dataset.
YColumn This property specifies which data array is going to be used as the Y coordinate in the generated polydata dataset.
ZColumn This property specifies which data array is going to be used as the Z coordinate in the generated polydata dataset.
