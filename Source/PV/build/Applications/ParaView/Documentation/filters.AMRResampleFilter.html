Resample AMR (AMRResampleFilter)

Converts AMR data to a uniform grid
This filter allows the user to specify a Region of Interest(ROI) within the AMR data-set and extract it as a uniform grid.
Property Description Default(s) Restrictions
Input This property specifies the input for this filter.

Accepts input of following types:

  • vtkOverlappingAMR
Demand-Driven Mode This property specifies whether the resampling filter will operate in demand-driven mode or not. 1

Accepts boolean values (0 or 1).
TransferToNodes This property specifies whether the solution will be transfered to the nodes of the extracted region or the cells. 1

Accepts boolean values (0 or 1).
NumberOfPartitions Set the number of subdivisions for recursive coordinate bisection. 1
