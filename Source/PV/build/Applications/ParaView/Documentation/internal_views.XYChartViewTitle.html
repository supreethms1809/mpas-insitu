XYChartViewTitle (XYChartViewTitle)

This is the base proxy for XY chart views.
Property Description Default(s) Restrictions
ChartTitle Set the title to use for the chart. Use ${TIME} to display the current time.
ChartTitleAlignment 1

The value(s) is an enumeration of the following:

  • Left (0)
  • Center (1)
  • Right (2)
ChartTitleFontFamily Set the chart title font family. Arial

The value(s) can be one of the following:

  • Arial
  • Courier
  • Times
ChartTitleFontSize Set the chart title font size. 14
ChartTitleBold Set the chart title bold. 0

Accepts boolean values (0 or 1).
ChartTitleItalic Set the chart title bold. 0

Accepts boolean values (0 or 1).
ChartTitleColor 0 0 0
