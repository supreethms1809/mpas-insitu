Unstructured NetCDF POP reader (UnstructuredPOPReaderCore)

Reads rectilinear grid data from a NetCDF POP file and converts it into unstructured data.
The reader reads regular rectilinear grid (image/volume) data from a NetCDF file and turns it into an unstructured spherical grid.
Property Description Default(s) Restrictions
FileName This property specifies the file name to read.

The value(s) must be a filename (or filenames).
Stride This property indicate the number of indices per dimension (i,j,k) to skip between each point included in this output. 1 1 1
VOI The data volume of interest (VOI). The VOI is a sub-extent of the data that you want loaded. Setting a VOI is useful when reading from a large data set and you are only interested in a small portion of the data. If left containing all 0's, then the reader will load in the entire data set. 0 -1 0 -1 0 -1
VerticalVelocity If there is a velocity vector with only 2 components, specify that the reader should compute the third component. 0

Accepts boolean values (0 or 1).
