TimeAnimationCue (TimeAnimationCue)

This is same as KeyFrameAnimationCue except that it has a new property "UseAnimationTime" which when set overrides the keyframes all together and directly uses the animation clock time to update the animated property.
Property Description Default(s) Restrictions
UseAnimationTime When enabled, the animated property is directly updated with the value of the Animation clock. Default value it true. 1

Accepts boolean values (0 or 1).
