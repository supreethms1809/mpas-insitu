ViewBase (ViewBase)

Property Description Default(s) Restrictions
ViewTime The pipeline update time for this view. This gets passed to all representations added to this view. none
CacheKey This is the cache key used to indentify which cache entry to use for rendering when caching is enabled. none
UseCache Indicates whether to use cache for subsequent renderings. 0

Accepts boolean values (0 or 1).
ViewPosition Set the position on this view in the multiview configuration. 0 0
ViewSize Set the size of this view in the multiview configuration. 400 400
Representations Representations in this view.
