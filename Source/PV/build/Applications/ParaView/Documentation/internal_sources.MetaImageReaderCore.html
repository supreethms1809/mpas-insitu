Meta Image reader (MetaImageReaderCore)

Reads binary UNC meta image data.
The Meta Image Reader reads binary UNC meta image data, and produces uniform rectilinear (image/volume) data. The default file extensions are .mhd and .mha.
Property Description Default(s) Restrictions
FileName This property specifies the file name for the Meta Image reader.

The value(s) must be a filename (or filenames).
