Programmable Source (ProgrammableSource)

Executes a user supplied python script to produce an output dataset.
This source will execute a python script to produce an output dataset. The source keeps a copy of the python script in Script, and creates Interpretor, a python interpretor to run the script upon the first execution.
Property Description Default(s) Restrictions
OutputDataSetType The value of this property determines the dataset type for the output of the programmable source. 0

The value(s) is an enumeration of the following:

  • vtkPolyData (0)
  • vtkStructuredGrid (2)
  • vtkRectilinearGrid (3)
  • vtkUnstructuredGrid (4)
  • vtkImageData (6)
  • vtkMultiblockDataSet (13)
  • vtkHierarchicalBoxDataSet (15)
  • vtkTable (19)
Script This property contains the text of a python program that the programmable source runs.
Script (RequestInformation) This property is a python script that is executed during the RequestInformation pipeline pass. Use this to provide information such as WHOLE_EXTENT to the pipeline downstream.
Parameters
PythonPath A semi-colon (;) separated list of directories to add to the python library search path.
