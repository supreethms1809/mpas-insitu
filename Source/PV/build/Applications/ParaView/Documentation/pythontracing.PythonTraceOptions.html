PythonTraceOptions (PythonTraceOptions)

Options proxy used to prompt user for tracing options when starting trace.
Property Description Default(s) Restrictions
PropertiesToTraceOnCreate Select which properties to save in trace 1

The value(s) is an enumeration of the following:

  • all properties (0)
  • any modified properties (1)
  • only user-modified properites (2)
FullyTraceSupplementalProxies Fully trace supplemental objects, such as color maps, color bars, etc. when accessed the first time in trace. 0

Accepts boolean values (0 or 1).
ShowIncrementalTrace Show trace as it is being generated. 0

Accepts boolean values (0 or 1).
