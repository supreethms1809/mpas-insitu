3D Text (VectorText)

3D geometric representation of a text string
The 3D Text source displays a text string as polygonal data.
Property Description Default(s) Restrictions
Text This property contains the text string to be displayed. The ASCII alphanumeric characters a-z, A-Z, and 0-9 are supported; so are ASCII punctuation marks. The only supported control character is "\n", which inserts a new line in the text string. 3D Text
